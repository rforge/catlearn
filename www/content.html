Introduction

Wills et al. (2016) provide a comprehensive introduction to the catlearn package. In brief:

Formal models are useful in psychology when they allow unambiguous comparison of the relative adequacy of theories across a broad range of phenomena (Wills & Pothos, 2012). However, the goal of broad comparison is not achievable by any one individual within a reasonable amount of time (Wills et al., 2016). Catlearn is designed to solve this problem by enabling efficient distributed collaboration. Catlearn is a framework and archive for formal modeling in psychology. It makes use of R's in-built documentation system to build an archival record of Canonical Independently Replicated Phenomena, an archive of formal models implemented as stateful list processors, and an archive of simulations. It is intended to supplement, rather than supplant, traditional publication routes. In particular, the documentation within catlearn is concise to the point of terseness, and does not obviate the need for more fulsome descriptions provided through traditional publication routes. Wills et al. (2016) provide a tutorial introduction to catlearn.

Installation

  1. Install the latest version of R from CRAN.
  2. (optional but recommended for those new to R) Install RStudio
  3. Install catlearn:
    • Stable version (choose this if unsure): From within R, type:

      install.packages('catlearn')

    • Unstable version (latest features): From within R, type:

      install.packages('catlearn', repos='http://R-Forge.R-project.org')

      NOTE: In order to install the unstable version on a Mac or a Linux machine, you will need to install additional software for building R packages from source. These are available on CRAN, in the same place you download R. Windows users do not need to worry about this.
